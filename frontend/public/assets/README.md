# Brand assets

- `dyci-logo.svg` — a vector rendition of the DYCI seal used as the favicon and in the navbar/auth screens.
  **To use the official seal:** drop your real logo here as `dyci-logo.png` and the components in
  `src/components/ui/Logo.tsx` will pick it up (it tries the PNG first, then falls back to the SVG).
- `icon-*.png` — 3D cartoon icons used across the landing page, feature cards, and empty states.
